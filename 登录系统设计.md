你是一个资深的前端开发工程师和 UI 设计师，精通现代 CSS 和极简美学。

【核心风格】
请帮我设计一个“黑白极简手账风（Minimalist Scrapbook）”的登录/注册页面。整体要求低饱和度、黑白灰配色、纯白背景、具有 Y2K/亚文化排版元素，以及纸质拼贴感。

【颜色规范】
- 页面背景：纯白色 #ffffff
- 卡片背景：纯白色，带极淡的弥散阴影 rgba(0,0,0,0.05)
- 边框与分割线：浅灰色 #eaeaea 或 #f0f0f0
- 文字颜色：主标题 #222，正文 #333，次要/装饰文字 #888 到 #ccc

【页面结构与组件细节】
- 主体容器：居中的手账本卡片，顶部带有下垂的“活页装订孔（Binder tabs）”设计。
- 头部信息区（左图右文）：
  - 背景：左侧底层有一个浅灰色的几何大五角星（clip-path 制作）。
  - 左侧：一个倾斜的“拍立得照片框”，带有阴影，左上角别着一个手绘风的“回形针”。照片内部用 SVG 图标占位。
  - 右侧：顶部有打字机风格的面包屑导航（如 Homepage > About Me），主标题带有颜文字，下方有英文签名。
- Tab 切换区：极简的纯文本 Tab（Login / Register），选中状态底部有黑色粗线条指示器。
- 表单区域：
  - 卡片容器：表单被包裹在一个带有灰色小标题栏（如 + Login Info，等宽字体）的线框卡片中。
  - 输入框：圆角药丸形状（border-radius: 20px），左侧带小图标，右侧带清除按钮，聚焦时边框变深。
  - 复选框：自定义方形复选框，选中时为黑底白勾，右侧文字分两行（主标题+英文小字副标题），功能为“显示密码”。
- 底部操作区：左侧为带下划线的“忘记密码/返回”链接，右侧为纯黑色的圆角药丸状“提交按钮”。

【排版与字体】
- 装饰性文字、标签、小标题必须使用等宽字体（Monospace），营造代码/打字机感。
- 整体保持大量留白，去除不必要的横线条纹，保持干净清爽。
